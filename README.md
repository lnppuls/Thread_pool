# 线程池
**尝试编写线程池**
## 新特性
### 1. c++11的并发编程，mutex锁与条件变量使用，主要条件变量的唤醒。
### 2. lambda表达式的使用，函数对象包装器function以及bind，对逻辑函数的包装。
### 3. 运用尾置返回类型和decltype推导类型。
### 4. 右值引用、移动语义以及完美转发。
### 5. 期物（Future），访问异步操作结果。
